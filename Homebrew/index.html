Homebrew

| 分类于 tools
Homebrew是macOS下的软件包管理器，类似于ubuntu的apt，是使用mac系统必备的开发者工具。

只要是能用 homebrew 安装的软件，就优先使用 homebrew 安装。一般来说使用 homebrew 安装的都是开发相关的软件，日常使用的比如微信之类的都是通过官网下载安装包或者在 app store 安装。

安装 Homebrew

访问Homebrew 官网获取安装命令。

安装完成后先执行以下命令安装 curl 和 wget。

常用命令

brew info这个命令很有用，可以显示出当前包的版本、安装路径、依赖包，以及一些提示信息，比如 Nginx 会提示你可以加到 brew services 中，PostgreSQL 会告诉你如何从旧版本迁移数据。这些信息会在包安装完成后自动显示，如果忘了的话可以用这个命令很方便地查看。

可以在/usr/local/Homebrew/Library/Taps这个路径下找到已经添加的第三方源仓库。

安装常用的包

不要卸载系统自带的 python2 和 ruby，可能导致系统自身某些功能因为缺失依赖而发生错误。

brew install python # 安装python3
brew install nodejs # 安装nodejs
brew install go # 安装go语言
brew install ruby # 安装最新版本的ruby

安装 docker

brew cask install docker

安装 JDK、Maven、Gradle

JDK

安装完毕后，在终端输入java -version即可看到正常输出。在 Intellij Idea 中，项目需要手动配置 JDK 路径，命令行输入/usr/libexec/java_home即可得到，通常是：/Library/Java/JavaVirtualMachines/<你安装的jdk版本>.jdk/Contents/Home。
