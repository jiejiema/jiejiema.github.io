Mac 安装MongoDB redis

| 分类于 soft
使用homebrew安装，便于维护

Homebrew是macOS下的软件包管理器，类似于ubuntu的apt，是使用mac系统必备的开发者工具。

安装

访问Homebrew 官网获取安装命令。
安装完成后先执行以下命令安装 curl 和 wget。

常用命令

安装MongoDB

安装redis 即 把 mongodb 替换为 redis

查看是否安装成功 which mongod
启动 mongod

使用

可视化工具 robo
https://robomongo.org/download

mongoose

操作mongodb 的工具 mongoose
https://mongoose.shujuwajue.com/
模式schemas 描述数据表，不需要手动创建表，用代码描述模型
schemaTypes
String; Number; Date; Buffer; Boolean; Mixed; Objectid; Array
自定义类型

模型models 由schemas发布生成， 对数据库增删改查，是由它操作

安装
npm i mongoose

对数据库配置
创建配置文件
创建文件夹 => 创建配置文件 config.js
在dbs文件夹中
export default ({
dbs: ‘mongodb://127.0.0.1:27017/dbs // 创建dbs 数据库地址127……
})

在dbs文件夹中建models文件夹
touch models/person.js （person.js文件名就是collection的名字）
import mongoose from ‘mongoose’
// 建表
let personSchema = new mongoose.Schema({
name: String, // collection内部数据
age: Number
})

// 由schema生成一个model 导出
// 由schema描述了数据库对应的collection（表），和模型（model）做关联
// expoer default mongoose.model(‘person’,personSchema) 可能报错
module.exports = mongoose.model(‘person’,personSchema)
app.js中 做连接
const mongoose = require(‘mongoose’)
const dbConfig = require(‘./dbs/config’)

// 服务器程序连接数据库
mongoose.connect(dbConfig.dbs, { // dbs为config中的变量dbs
useNewParser: true
})
